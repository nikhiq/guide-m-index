# guide-m-index

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不针对特定域名或具体网站，仅作为通用页面存档与索引使用。

## 项目简介

本仓库的主要目的是集中管理和展示一组独立的 HTML 页面。每个页面均为自包含的静态文件，内容涵盖信息展示、工具界面、文档参考等不同用途。仓库本身不运行任何后端服务，也不依赖外部数据库，所有内容均通过浏览器直接访问。

## 目录结构

```
├── index.html          # 仓库入口页面，列出所有归档页面链接
├── pages/              # 存放所有独立的 HTML 页面文件
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的导航页面，以目录形式列出所有归档页面，方便快速跳转。
- `pages/`：所有独立 HTML 页面均存放于此目录下，文件名应具有描述性。

## 页面归档说明

每个 HTML 页面均为独立文件，不依赖外部资源（除必要的公共库或样式外）。页面内容可以是信息展示、简单工具、文档片段等。所有页面均保持静态，不涉及用户交互数据收集或动态内容生成。

归档页面遵循以下原则：
- 每个页面自包含，可脱离本仓库单独运行。
- 页面之间无相互依赖关系。
- 页面内容应当清晰、简洁，符合通用阅读习惯。

## 维护说明

本仓库由维护者不定期更新。新增页面时，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加对应条目。若需修改或删除页面，请同步更新索引文件。

欢迎提交 Issue 或 Pull Request 来提议新增、修改或删除页面。所有变更应保持页面独立性与内容通用性。
